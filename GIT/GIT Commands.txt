Create new Branch, new file and move that to old branch
========================================================

1) git checkout -b NewFolder     
   Switched to a new branch 'NewFolder'

2) git branch  
  * NewFolder
   master

3) git commit -am "create new branch"
   On branch NewFolder
   Untracked files:
  (use "git add <file>..." to include in what will be committed)
        banner.png


4) git push -u origin NewFolder

 * [new branch]      NewFolder -> NewFolder
branch 'NewFolder' set up to track 'origin/NewFolder'.
